# DiabloIIIMarco
Tool bảo vệ chuột và bàn phím cho Diablo III

Phát triển từ code gốc của: https://github.com/langmaninternet/DialoIIISupport

Các tính năng chính
------------
- F1: Tự động bấm chuột trái
- F3: Tự động bấm chuột phải
- F2: Tự động bấm Hotkey từ 1=>4
